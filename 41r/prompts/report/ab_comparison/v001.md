## 추가: A/B 비교 분석
두 variant의 세션 데이터를 비교하여:
1. 어떤 variant가 더 나은 사용자 경험을 제공하는지
2. known_result(실제 A/B 테스트 결과)가 주어지면, 페르소나 예측과 일치하는지 역검증
3. 일치하면 왜, 불일치하면 왜 차이가 나는지 분석

출력에 "ab_comparison" 필드를 추가:
```json
{
  "ab_comparison": {
    "predicted_winner": "A|B",
    "confidence": "높음|보통|낮음",
    "reasoning": "근거",
    "matches_known_result": true|false|null,
    "discrepancy_analysis": "불일치 시 원인 분석"
  }
}
```
