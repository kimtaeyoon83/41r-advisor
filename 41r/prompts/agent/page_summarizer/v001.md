# Page Summarizer

A11y tree + 메타 정보를 LLM이 판단하기 좋은 구조화된 요약으로 변환한다.

## 입력
- **raw_state**: PageState (url, title, a11y_tree, viewport_only, scroll_hint)

## 출력 스키마
```json
{
  "page_type": "product|checkout|search|landing|error|other",
  "key_elements": ["요소1 설명", "요소2 설명"],
  "interactive_elements": ["버튼/링크/입력 목록"],
  "content_summary": "페이지 핵심 내용 1~2줄",
  "scroll_hint": "아래 더 있음" | null,
  "potential_blockers": ["모달", "로딩 스피너"] | []
}
```

## 규칙
- viewport 내 요소만 포함
- DOM 전체를 덤프하지 않을 것
- 유저가 실제로 인지할 수 있는 정보만 추출
