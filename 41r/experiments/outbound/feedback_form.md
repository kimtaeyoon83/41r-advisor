# 41R CPO 피드백 수집 폼

> **용도**: 데모/미팅 후 유료 전환 의향 및 제품 피드백 수집
> **형식**: Google Form 또는 Typeform으로 변환
> **소요 시간**: 3분

---

## 섹션 1: 기본 정보

1. **이름** (텍스트)
2. **회사명** (텍스트)
3. **직책** (드롭다운)
   - CPO / VP Product
   - Head of Growth
   - Product Manager
   - UX Researcher
   - 기타: ___

4. **현재 A/B 테스트 빈도** (단일 선택)
   - 주 1회 이상
   - 월 2~4회
   - 월 1회 이하
   - 거의 안 함
   - A/B 테스트를 하고 싶지만 못 하고 있음

---

## 섹션 2: 현재 Pain Point

5. **A/B 테스트에서 가장 큰 어려움은?** (복수 선택)
   - [ ] 결과 나올 때까지 시간이 너무 오래 걸림
   - [ ] 충분한 트래픽이 없어 통계적 유의성 확보 어려움
   - [ ] 테스트 설계/가설 수립이 어려움
   - [ ] 엔지니어링 리소스 부족
   - [ ] 결과 해석이 어려움
   - [ ] 비용이 너무 높음
   - [ ] 기타: ___

6. **A/B 테스트 결과를 기다리느라 출시가 지연된 경험이 있나요?** (단일 선택)
   - 자주 있다
   - 가끔 있다
   - 거의 없다
   - A/B 테스트를 안 해서 해당 없음

---

## 섹션 3: 41R 리포트 평가

7. **오늘 보신 41R 리포트의 유용성은?** (1~5점)
   - 1: 전혀 유용하지 않음
   - 5: 매우 유용, 당장 활용 가능

8. **리포트에서 가장 가치 있었던 부분은?** (복수 선택)
   - [ ] 세그먼트별 행동 분석 (충동형 vs 신중형)
   - [ ] UX 문제 자동 발견
   - [ ] 전환 퍼널 분석
   - [ ] 구체적 개선 제안
   - [ ] 24시간 내 결과 (속도)
   - [ ] 기타: ___

9. **리포트에서 부족하거나 추가되면 좋겠는 부분은?** (자유 텍스트)

---

## 섹션 4: 유료 전환 의향 (핵심)

10. **이 서비스가 정식 출시되면 유료로 사용할 의향이 있나요?** (단일 선택)
    - 확실히 사용할 것이다
    - 아마 사용할 것이다
    - 잘 모르겠다
    - 아마 사용하지 않을 것이다
    - 확실히 사용하지 않을 것이다

11. **월 얼마까지 지불할 의향이 있나요?** (단일 선택)
    - $0 (무료만 사용)
    - $99/월 이하
    - $100~299/월
    - $300~499/월
    - $500~999/월
    - $1,000/월 이상

12. **가장 먼저 41R로 테스트해보고 싶은 것은?** (자유 텍스트)
    - 예: "랜딩 페이지 리디자인", "체크아웃 플로우 개선", "가격 페이지 최적화"

---

## 섹션 5: 다음 단계

13. **파일럿 테스트에 참여할 의향이 있나요?** (단일 선택)
    - 예, 바로 시작하고 싶다
    - 예, 하지만 좀 더 알아보고 싶다
    - 아직 결정하지 못했다
    - 아니오

14. **추가로 나누고 싶은 이야기가 있나요?** (자유 텍스트)

## 분석 기준

### H1 Kill Criteria 매핑

| 질문 | H1 판단 기준 |
|---|---|
| Q10 (유료 의향) | "확실히" + "아마" ≥ 30명 중 5명 = 16.7% → **5% 이상이면 H2 진행** |
| Q11 (가격) | $300+/월 응답이 3명 이상이면 B2B SaaS 가격 가능 |
| Q13 (파일럿) | "바로 시작" 2명 이상이면 첫 유료 고객 후보 |

### 응답 우선순위
- Q10 + Q11: 유료 전환 가능성 (H1 핵심)
- Q8: 가치 인식 포인트 (마케팅 메시지 최적화)
- Q5: 시장 pain point 확인 (포지셔닝)
- Q12: 첫 유스케이스 (제품 방향)
