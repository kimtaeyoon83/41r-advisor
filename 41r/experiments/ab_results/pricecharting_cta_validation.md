# A/B 역검증 #1 — PriceCharting CTA 버튼 텍스트

> **검증일**: 2026-04-13
> **출처**: VWO Success Story (https://vwo.com/success-stories/pricecharting/)

---

## 실제 A/B 테스트 결과

| 항목 | 내용 |
|---|---|
| 회사 | PriceCharting |
| 테스트 대상 | CTA 버튼 텍스트 |
| Variant A (Control) | "Download" 버튼 + 가격 노출 |
| Variant B (Winner) | "Price Guide" 버튼 + 가격 제거 |
| 결과 | **+620.9% CTR** (Variant B 승) |
| 신뢰도 | 99.9%, 1주 내 결론 |
| 핵심 인사이트 | "Download"보다 "Price Guide"가 사용자 의도와 더 정확히 매칭. 가격 노출이 오히려 클릭을 억제 |

---

## 41R 페르소나 세션 결과

### p_impulsive (충동적 쇼퍼) — 3턴, 이탈

**행동 요약**: 히어로 영역 스캔 → 시각적 CTA 없음 확인 → scroll 1회 → 시각적 트리거 전무 판단 → close_tab

**CTA 관련 관찰**:
- Turn 1: "컬러 CTA 버튼, 할인 배너, 가격 숫자 등 반응할 시각적 요소가 발견되지 않음"
- Turn 2: "텍스트 링크와 카테고리 목록 위주 구조"
- Turn 3: 시각적 트리거 전무 → 즉시 이탈

**판정**: 충동형 유저는 CTA 자체를 발견하지 못함. "Download" 든 "Price Guide"든 **시각적으로 눈에 띄지 않으면** 도달조차 안 함.

### p_cautious (신중한 리서처) — 10턴, 최대턴 소진

**행동 요약**: 서비스 설명 read 2회 → scroll → read → navigate(카테고리) → Download Price Guide 섹션 read 2회(실패) → scroll 2회(실패) → 재로드 시도

**CTA 관련 관찰**:
- Turn 5: A11y tree에서 카테고리 진입 시도
- Turn 6: "Download Price Guide" heading 발견, read 시도
- Turn 7: 같은 섹션 재시도 — diff 공백으로 내용 확인 실패
- 10턴 소진 후에도 **다운로드 조건(무료/유료, 파일 형식)을 확인하지 못함**

**판정**: 신중형 유저는 "Download Price Guide"까지 도달했으나, 정보 부족으로 클릭 결정을 못 함.

## 역검증 대조

### 실제 A/B 결과와 일치하는 부분

| 실제 결과 | 페르소나 예측 | 일치 여부 |
|---|---|---|
| "Download" 텍스트가 클릭을 억제 | p_cautious가 "Download Price Guide" 도달 후에도 클릭하지 않음 — 다운로드 조건 불명확 | **일치** |
| "Price Guide"가 더 의도 매칭 | p_cautious의 태스크가 "가격 가이드 다운로드"인데, "Price Guide"라는 텍스트가 더 직관적으로 매칭됨 | **일치 (추론)** |
| 가격 제거가 CTR 증가 | p_impulsive가 가격 숫자 부재를 이탈 이유로 지목 — 그러나 이건 CTA 옆 가격이 아닌 사이트 전반의 가격 부재 | **부분 일치** |

### 불일치하는 부분

| 실제 결과 | 페르소나 예측 | 차이 원인 |
|---|---|---|
| CTA 클릭률 620% 증가 | 두 페르소나 모두 CTA 클릭 0회 | 페르소나가 CTA에 **도달하기 전에 이탈**(충동형) 또는 **기술적 문제**(신중형). A/B 테스트는 CTA까지 도달한 유저 기준, 41R은 전체 여정 기준 |
| B variant가 명확히 우세 | 41R은 A/B variant를 직접 비교하지 않음 | 현재 실행은 현재 사이트(단일 variant)에 대한 분석. A/B 비교를 위해서는 두 variant를 별도로 구성해야 함 |

### 한계

1. **단일 variant만 테스트**: 현재 PriceCharting 라이브 사이트만 방문. A/B의 두 variant를 모두 테스트하지 않음.
2. **CTA 미도달**: 충동형은 CTA까지 도달 전 이탈, 신중형은 기술적 문제로 CTA 상호작용 실패.
3. **환경 변수**: A11y tree 품질 문제가 신중형 유저의 행동을 왜곡. 실제 유저는 화면을 볼 수 있으므로 이 문제 없음.

## 결론

**부분 검증 성공**. 페르소나의 행동 패턴(충동형 즉시 이탈, 신중형 정보 탐색 후 미결정)은 실제 A/B 결과의 방향성과 일치하지만, **정량적 역검증(620% CTR 차이 재현)은 불가능**했음.

**원인**: 현재 시스템은 라이브 사이트 단일 방문만 가능. A/B 역검증을 위해서는:
1. 두 variant의 HTML을 로컬에 구성하여 통제된 환경에서 테스트
2. 또는 Wayback Machine 등에서 과거 variant 복원

**H1 데모 활용**: "우리 시스템이 CTA 디자인의 문제(시각적 부재, 정보 불충분)를 자동 발견했다"는 사실 자체가 가치 증명. 정량 재현은 H2 단계 과제.
