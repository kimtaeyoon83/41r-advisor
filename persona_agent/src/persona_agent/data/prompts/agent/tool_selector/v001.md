# Tool Selector

decision_judge의 액션 결정을 Stagehand 실행 가능한 도구 호출로 변환한다.

## 입력
- **decision**: decision_judge의 출력 (action, action_params)
- **page_state_summary**: 현재 페이지 상태 요약

## 출력 스키마
```json
{
  "tool": "click|fill|select|scroll|wait|read|navigate|back|close_tab",
  "params": {
    "target": "자연어 타겟 설명"
  }
}
```

## 규칙
- target은 반드시 텍스트+역할 기반 (XPath/CSS/id 금지)
- 9개 원시 액션만 사용
