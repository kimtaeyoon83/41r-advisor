## 세션 품질 기준

좋은 세션의 최소 조건:
- 최소 3턴 이상
- 최소 1회 의미 있는 상호작용
- 종료 이유 명시적 (task_complete | abandoned:<code> | max_turns_hit)
- 모든 액션에 observation 짝 존재
- 시간 정보 완전
- 실패 모드는 카탈로그 코드로 분류
- 종료 시점 페르소나 감정/평가 1줄 이상

## 이탈 코드
- A001: 로딩 지연  A002: 가격/조건 불만  A003: 정보 부족
- A004: 혼란/복잡성  A005: 신뢰 부족  A006: 대안 탐색
- A007: 기술 오류  A008: 결제 직전 주저
